# **ResearchVerse [CSE370 Project]**

**Main Features:**

* peer finding based on research interest
* collab request
* Take appointment + paper review (advisor/expert)
* upload resources for beginners (published conference papers, upcoming conferences etc.) 
* Community
* wishlist
* progress tracker 

**Language Used:**
* Frontend: HTML, CSS 
* Backend: PHP
* Database: MySQL
